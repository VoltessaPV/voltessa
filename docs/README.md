# Voltessa

Voltessa е платформа за автоматизация на фотоволтаични централи.

Основната цел е автоматично управление на производството според пазарните цени на електроенергията, така че собственикът на централата да минимизира загубите и да максимизира приходите.

## Първа версия (MVP)

Поддържа:

- Huawei FusionSolar
- Една централа
- Автоматично спиране на експорта
- Автоматично възстановяване на експорта
- Конфигурируеми прагове
- Event Log

## Основен принцип

Voltessa не заменя SCADA системата.

Voltessa добавя интелигентен слой за автоматизация върху съществуващите платформи.
