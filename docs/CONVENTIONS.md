# Coding Conventions

## Architecture

Controllers съдържат само HTTP логика.

Services съдържат бизнес логика.

Drivers комуникират с производителите.

Clients комуникират с външни API.

## TypeScript

Използвай:

import type

за interfaces и types.

## Development

Една User Story = един Commit.

След всяка завършена функционалност се обновява документацията.

Не се добавят временни решения.

Архитектурните решения се описват като ADR.
